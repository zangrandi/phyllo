# Phyllo Ruby Gem

A Ruby interface for Phyllo API.

## Installation

    $ gem install phyllo

## Configuration

Create a configuration file

```ruby
Phyllo.configure do |config|
  config.api_key = "xxx"
  config.sandbox = false
end
```
